# FitEats 健身食谱 Web App 规范

## 1. Concept & Vision

**FitEats** — 一款清新风格的健身食谱应用，介于"冰冷健身平台"和"花哨美食app"之间。有健身的专业感，有饮食的温暖感，还有抽卡的惊喜感。像一个懂生活的健身教练，而不是严肃的营养师。

**核心体验**：选目标→抽卡→做菜→打卡，像玩游戏一样做健身餐。

## 2. Design Language

### 2.1 色彩系统
```
主色（健康绿）: #2ECB71
辅助色（食欲橙）: #FF7A45  
强调色（科技蓝）: #3498DB
背景色（浅灰底）: #F5F7FA
深色文字: #1A1A2E
次要文字: #6B7280
白色: #FFFFFF
```

### 2.2 字体
- **标题/功能**: "Noto Sans SC", sans-serif (Bold 700)
- **正文**: "Noto Sans SC", sans-serif (Regular 400)
- **数字强调**: "Oswald", sans-serif (Bold)

### 2.3 空间系统
- 基础单位: 8px
- 卡片圆角: 8px
- 页面内边距: 16px
- 卡片间距: 12px

### 2.4 动效哲学
- 卡片hover: scale(1.02) + shadow增强, 200ms ease-out
- 按钮点击: scale(0.96) + 50ms
- 抽卡弹窗: 从底部滑入 + 弹性缩放
- 标签切换: 颜色过渡 150ms

## 3. Layout & Structure

### 3.1 页面架构（移动端优先, 375px基准）

**底部导航（4个Tab）:**
1. 🏠 首页（菜谱卡片流）
2. 🎲 抽卡（随机食谱）
3. 📋 采购（清单管理）
4. 👤 我的（个人中心）

### 3.2 页面结构

**首页 Tab:**
- 顶部：目标筛选标签（减脂/增肌/塑形）+ 时段筛选（早餐/午餐/晚餐/加餐）
- 中间：菜谱卡片流（垂直滚动）
- 卡片结构：标签区→图片区→营养数据区→功能区

**抽卡 Tab:**
- 顶部：目标选择（3个圆形按钮）
- 中间：时段选择（横向4个标签）
- 底部：主按钮「一键抽卡」
- 弹窗：抽卡结果卡片（带惊喜动效）

**采购 Tab:**
- 顶部：页面标题 + 全选操作
- 中间：分类食材列表（肉类/蔬菜/主食/调料）
- 每行：复选框 + 食材名 + 克数
- 底部：已购统计

**我的 Tab:**
- 用户信息卡片
- 数据统计（打卡天数/解锁菜谱/体重变化）
- 功能列表（我的收藏/我的打卡/身体数据）

## 4. Features & Interactions

### 4.1 菜谱卡片
- **点击卡片**：展开详情弹窗（显示完整食材清单+步骤）
- **收藏按钮**：切换星星填充状态 + Toast提示
- **筛选标签**：点谁谁高亮，实时过滤卡片

### 4.2 抽卡系统
- **选择目标**：点击圆形按钮选中（边框变粗+图标放大）
- **选择时段**：点击标签选中（背景色填充）
- **点击抽卡**：
  1. 按钮禁用1秒
  2. 骰子旋转动画
  3. 结果卡片从底部滑入
- **结果卡片**：显示完整菜谱信息 + 底部「加入今日食谱」按钮
- **关闭弹窗**：点击遮罩层或关闭按钮

### 4.3 采购清单
- **勾选食材**：行内划线效果 + 计数更新
- **分类展示**：同类合并，清晰分组
- **一键清空**：全部取消勾选

### 4.4 打卡功能
- **打卡按钮**：点击打卡 + 连续天数累加
- **打卡成功**：Confetti动效 + Toast提示

## 5. Component Inventory

### 5.1 菜谱卡片 (RecipeCard)
- 尺寸：宽度100%（页面宽度-32px），高度auto
- 状态：默认、hover（放大+阴影）、已收藏（星星填充）
- 包含：标签（减脂/增肌/塑形 + 时段）、图片（高度200px）、营养数据条、功能按钮

### 5.2 抽卡按钮 (抽卡页主按钮)
- 尺寸：宽度80%，高度56px
- 样式：科技蓝渐变背景，白色粗体字，左侧骰子图标
- 状态：默认、hover（亮度+10%）、点击（缩放0.96）、禁用（灰色，loading动画）

### 5.3 抽卡结果弹窗 (抽卡结果卡片)
- 尺寸：宽度90%，最大高度85vh，居中弹出
- 样式：白色背景，顶部渐变边框，卡片放大动画
- 包含：关闭按钮、欢迎语、完整菜谱信息、食材替换提示、操作按钮

### 5.4 营养数据条 (NutritionBar)
- 布局：横向4等分，每个数据块包含图标+数字+单位
- 图标：火焰(卡路里)、肌肉(蛋白)、面包(碳水)、牛油果(脂肪)
- 数字：Oswald Bold 20px 主色

### 5.5 底部导航 (TabBar)
- 高度：60px + safe-area
- 4个Tab：图标+文字，选中高亮主色
- 状态：未选中（灰色图标+文字）、选中（主色图标+加粗文字）

### 5.6 筛选标签 (FilterTag)
- 尺寸：高度32px，内边距12px
- 状态：未选中（浅灰背景+深色文字）、选中（对应颜色背景+白色文字）

### 5.7 采购项 (ShoppingItem)
- 布局：左侧圆形复选框 + 食材名 + 右侧克数
- 状态：未选中、已勾选（文字划线+透明度50%）

### 5.8 数据卡片 (StatCard)
- 布局：图标 + 数字 + 标签
- 数字：Oswald Bold 24px 主色

## 6. Technical Approach

### 6.1 技术栈
- **HTML5** + **CSS3** + **Vanilla JavaScript**
- 单文件应用（SPA风格）
- 无框架依赖，直接浏览器运行

### 6.2 CSS方案
- CSS变量管理主题色
- Flexbox布局
- CSS动画（keyframes）
- 媒体查询响应式

### 6.3 数据
- 模拟数据（硬编码JSON）
- LocalStorage持久化（收藏、购物车、打卡记录）

### 6.4 交互
- Tab切换无刷新
- 弹窗用fixed遮罩层
- 触摸事件优化
